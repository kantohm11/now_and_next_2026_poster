# Poster Plan: iTHEMS Now & Next 2026

## Format
- A0 poster (841 × 1189 mm)

## Overview
Poster presentation version of the span_enforcing slides (originally Quarto slides for Simons Collaboration Annual Meeting 2025).

## Audience
- Broader audience including mathematical biologists (and other non-specialists)
- Need to balance accessibility with technical depth

## Structure
- **First half: Pedagogical introduction**
  - Accessible to broad audience
  - Motivate the problem and key ideas
- **Second half: Technical content**
  - Main logic and examples
  - Skip details on module categories — focus on the core argument and illustrative examples
